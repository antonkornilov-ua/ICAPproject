# Project Description
This project was developed as a test assignment to demonstrate the skills and knowledge of the developer. 

## Technologies Used
The following technologies were utilized in the development of this project:

* React
* Tailwind
* Typescript

## Additional npm Packages Installed
The following npm packages were additionally installed to enhance the functionality of the application:

* react-router-dom
* react-icons

## Fully Responsive Design
The application is designed to be fully responsive, ensuring optimal user experience across different devices and screen sizes.

## Dynamic Quotes Section
The project includes a dynamic quotes section where data is loaded from a server. This section provides users with a dynamic and engaging experience by displaying a variety of quotes.

## Custom Card Wrappers
To enhance the visual appeal and functionality of the cards within the application, custom card wrappers were created from scratch. These wrappers provide a consistent and aesthetically pleasing design for the cards throughout the project.
